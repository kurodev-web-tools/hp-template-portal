主張を、反応する導線まで落とし込む。

VIVID IMPACT はコピーだけを作るのではなく、主張がどこで見つかり、どう比較され、どの接点で反応が生まれるかまでを設計します。依頼内容が粗くても、必要な接点と納品物をこちらで整理します。

相談の入口

CLAIM,
FLOW,
LAUNCH.

新サービス発表、採用ページ刷新、キャンペーン整理など、相談の言葉が粗くても構いません。まずは何を揃えるべきかから返します。

ブランドや事業の核になる一文を定義し、すべての接点の判断基準にします。

  • 対象: ブランド刷新 / 新規事業 / 採用訴求
  • 成果物: ブランドメッセージ設計書、コピーライン、言い換えルール
  • 進め方: ヒアリング → 競合整理 → 主張設計
  • 向いている相談: 言いたいことはあるが伝わり方が散っている状態

KV、コピー、動画、投稿、広告見出しをひとつの主張で束ねます。

  • 対象: 新サービス告知 / 期間キャンペーン / ブランド刷新
  • 成果物: KV、動画構成、投稿文、配信素材、告知見出し
  • 進め方: 企画整理 → 表現設計 → 展開素材制作
  • 向いている相談: 各媒体で言っていることがズレている状態

大見出し主導で、第一印象から問い合わせまでの流れを整理します。

  • 対象: LP / サービスサイト / 採用ページ
  • 成果物: 情報設計、見出し群、構成案、CTA導線設計
  • 進め方: 読了導線設計 → ページ構成 → 実装整理
  • 向いている相談: 印象はあるが行動につながりにくい状態

発表順、説明順、配信素材、資料を一気に整えて、公開時のズレを防ぎます。

  • 対象: サービス発表 / IR補助 / 採用広報 / イベント告知
  • 成果物: 発表シナリオ、配布資料、告知文、説明順
  • 進め方: 公開計画整理 → 接点洗い出し → 素材制作
  • 向いている相談: 発表タイミングは決まっているが見せ方が弱い状態
納品物の例

deliverable / launch set

公開前72時間の導線調整

トップ見出し、リリース冒頭文、営業メール導入文、SNS告知の順番を合わせて、ローンチ直前に一気に最終調整します。

deliverable / message pack

ブランドメッセージ設計書

主張の一文、支える見出し、言い換えルール、媒体別の使い分けをまとめた運用用ドキュメントです。

進行イメージ
  1. 01 Input事業内容、公開時期、今ある資料、競合の見え方を共有いただきます。
  2. 02 Direction一文目と接点ごとの役割を整理し、優先順位を切ります。
  3. 03 Production見出し、LP、資料、投稿、告知素材へ一気に展開します。
  4. 04 Release公開順と説明順を最終調整し、直前差し込みまで対応します。
よくある依頼の組み合わせ
新規SaaS立ち上げ: 命名 + LP + 発表資料 + 営業導入文
採用訴求刷新: 採用主張 + 採用LP + 求人票 + 説明会冒頭文
キャンペーン整理: KV + 広告見出し + 投稿文 + 特設LP
