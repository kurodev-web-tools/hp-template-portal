Support Formats

Support Formats

案件の性質ごとに、必要な構成と成果物が変わるため、3つの形式に分けて相談を整理しています。

01

Launch Program

ブランドや新サービスの初回公開向け。要点整理、提供範囲、CTA までを一枚でつなぎます。

02

Campaign Flight

期間限定施策や募集訴求向け。比較軸、限定情報、応募導線を読みやすく並べます。

03

Event Runway

イベントやポップアップ向け。開催概要、会場情報、参加導線を同じ熱量でまとめます。

Deliverables Overview

Deliverables Overview

成果物は演出名ではなく、公開や相談に必要な単位で揃えています。

Page Core Hero / Intro / CTA

最初に何を伝えるか、誰向けか、次にどう動けばいいかを固定します。

Offer Detail Scope / Menu / Use Cases

支援範囲、向いている案件、相談時に揃えたい情報を見せます。

Proof Set Metrics / Assets / Interview

写真、実績、コメント、開催概要など、信頼につながる材料を整理します。

Contact Setup Form / Timing / Notes

返信目安、入力項目、初回相談に必要な情報を明文化して着地させます。

Project Flow

Project Flow

初回相談から公開準備までを4段階で固定し、どの案件でも判断がぶれないようにします。

01

Share The Brief

公開予定日、主役にしたい内容、使える素材を共有してもらいます。

02

Outline The Page

必要セクション、訴求順、補足情報の深さを決めて、ページ骨格を固めます。

03

Shape The Visuals

写真やトーンを整理し、見出し、カード、CTA の強弱を合わせます。

04

Prep For Launch

スマホ表示、問い合わせ導線、公開前の表記を確認して公開状態へ仕上げます。

Ready To Share The Brief

Ready To Share The Brief

案件の種類が近ければ、まずは公開予定日と主役にしたい内容を共有してください。必要な素材やページ構成は、初回相談の中で一緒に整理できます。
