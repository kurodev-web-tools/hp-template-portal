Night formats

Projects built for after dark

このページでは、イベント運営だけでなく、空間演出、映像、ブランド接点までを扱う構成にしています。サービス一覧ではなく、夜の体験をつくる単位で整理しています。

Format 01

Event direction

出演構成、空間導線、サイン計画、VIP 導線、撮影ポイントまで含めて一夜の流れを設計します。

Format 02

Spatial light

会場の照明、LED、アーチ、導線演出をまとめてデザインし、夜の印象を更新します。

Format 03

Motion package

ティザー映像、会場スクリーン、ショート動画、アフタームービーを統一トーンで制作します。

Format 04

Brand collision

ブランドコラボ、ポップアップ、商業施設ナイトイベントなど共同企画向けの設計です。

Process

How projects move through the night

01 / Pulse check

Read the brief

ブランド、会場、客層、深夜帯の行動を整理し、企画の芯を定めます。

02 / Build the scene

Shape the route

会場導線、ライティング、スクリーン演出、撮影導線を図面化します。

03 / Launch

Run the night

現場運営、出演者連携、配信やリアルタイム演出まで一括で支えます。

04 / After glow

Extend the impact

終了後にアーカイブ、短尺素材、次回告知へ接続し、熱量を残します。

Engagement options

Choose the right signal depth

価格や実績保証は入れず、どの深さまで伴走するかで分けています。公開前に自社の支援範囲へ書き換えてください。

  • 体験設計の方向づけ

Direction lane

Full night build

  • 企画と演出の統合ディレクション

  • 当日運営とアフター導線

Partnership lane

District program

  • 複数拠点を横断する回遊企画
  • ブランドと施設の協業設計
  • 年間施策の編集
