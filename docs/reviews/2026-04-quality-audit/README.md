# 2026-04 Quality Audit

このフォルダは `public/templates` の公開品質監査結果をまとめるための置き場です。

## Structure
- `summary.md`: 監査全体の統合メモ
- `codex/`: Codex 側の監査結果
- `kimi/`: Kimi 側の監査結果

## Recommended Flow
1. まずカテゴリ単位で監査結果を書く
2. 問題が多いテンプレートだけ個別メモを追加する
3. 最後に `summary.md` で統合して優先順位を決める

## Default Files
- `business.md`
- `lp.md`
- `portfolio.md`
- `streamer.md`

必要になった場合のみ `templates/` などを追加してテンプレート単位の深掘りを行う。
